## 🐦 X (Twitter) 運用

### 初期設定（1回のみ、15分）
1. [developer.x.com](https://developer.x.com/) でアプリ作成
2. **Free tier** を選択（月1,500ツイートまで無料）
3. 以下の4つのキーを取得:
   - API Key
   - API Secret
   - Access Token
   - Access Token Secret
4. GitHubの Settings → Secrets に追加:
   ```
   X_API_KEY=取得した値
   X_API_SECRET=取得した値
   X_ACCESS_TOKEN=取得した値
   X_ACCESS_SECRET=取得した値
   ```
5. 完了！以降は毎日朝8時・夜8時に自動投稿される

### 自動で動くこと
- 毎週月曜: AIが1週間分の投稿を自動生成（14投稿）
- 毎日8:00 / 20:00: キューから自動投稿
- 投稿内容: Tips(40%) / 開発ログ(30%) / 商品PR(20%) / 交流(10%)

### あなたがやること（毎日5分）
1. **通知を見る** — リプが来ていたら返事（1-2件でOK）
2. **引用RT** — 共感できるツイートがあれば引用（週1-2回）
3. **フォロー** — #Obsidian #個人開発 で検索して同じ界隈の人をフォロー（週5人）

### コツ
- 自動投稿は「情報発信」、手動は「交流」に集中
- 押し売りしない。9割は価値提供、1割は宣伝
- 「AIで自動化してます」とは言わない

## 📝 note.com 運用

### 初期設定（1回のみ、10分）
1. [note.com](https://note.com/) でアカウント作成
2. プロフィール設定:
   ```
   名前: AbsCL
   自己紹介: AI × 自動化が好きな個人開発者。
   Obsidian/Python/Discord Bot/GCPを使って何でも自動化します。
   ```
3. プロフィール画像を設定
4. 「マガジン」を作成:
   - 名前: 「AI × 個人開発 マネタイズ日記」
   - 説明: 「AIと自動化でパッシブインカムを構築する過程を全公開。」
   - 月額: **¥500**（まずは安めから）

### 記事の出し方

#### 単品記事（¥300〜500）
ドラフトは全部 `content/` に用意済み:

| ファイル | 内容 | 価格 |
|:---|:---|:---|
| `note-article-draft.md` | AIで1日で商品を作った話 | ¥300 |
| `note-article-series.md` の記事2 | Obsidian設計パターン | ¥500 |
| `note-article-series.md` の記事3 | GCP無料枠サーバー | ¥300 |
| `note-article-series.md` の記事4 | Discord Bot AI搭載 | ¥300 |

#### 投稿手順（毎回5分）
1. note.com で「テキスト」→「投稿」
2. 上記ファイルから**内容をコピペ**
3. 「---有料ライン---」の位置で**有料エリアを設定**
4. 価格を設定（¥300 or ¥500）
5. ハッシュタグ追加: `#AI` `#個人開発` `#Obsidian` `#Python`
6. 公開

#### ペース
- **最初の1ヶ月**: 週1本ペース（4本出す）
- **2ヶ月目以降**: 月2本ペース
- マガジン購読者が10人超えたら月額を¥980に改定

### コンテンツの作り方（将来）
新しい記事が必要になったら:
```

「○○についてnote記事のドラフトを書いて」
→ ドラフトが content/ に生成される
```

## 💰 アフィリエイト運用

### 初期設定（1回のみ、20分）

#### Amazon アソシエイト
1. [affiliate.amazon.co.jp](https://affiliate.amazon.co.jp/) で申請
2. サイト情報: ブログのURLを登録
3. **審査通過に3記事以上のコンテンツが必要**（ブログに8本あるのでOK）
4. 審査通過後、アソシエイトIDを取得
5. 以下のファイルのリンクを更新:
   - `blog/articles/*.html` 内の技術書リンク
   - `content/affiliate-strategy.md` のリンク

#### Render アフィリエイト
1. [render.com/affiliates](https://render.com/affiliates) で申請
2. 紹介リンクを取得
3. GCPの記事に「代替PaaS」としてリンク設置

#### もしもアフィリエイト（包括的）
1. [af.moshimo.com](https://af.moshimo.com/) で登録
2. Amazon/楽天/Yahoo!ショッピングを一括で使える
3. W報酬制度（追加12%ボーナス）
4. サイト審査あり（ブログURLで通る）

### アフィリエイトリンクの設置場所

全部の記事に既にプレースホルダーがあるので、IDを取得したら差し替えるだけ:

```
ブログ記事 → 関連ツール・書籍のリンク
LP → 「使用技術」セクション
README → 推奨ツールのリンク
```

### あなたがやること
- **審査通過まで**: 何もしなくてOK（ブログに記事があるので申請するだけ）
- **通過後**: Antigravityに「アフィリエイトID取得した、リンクを全部差し替えて」と頼む

## 🆘 困ったとき

| 状況 | 対処 |
|:---|:---|
| X自動投稿が止まった | GitHub Actions のログを確認。APIキー期限切れの可能性 |
| ブログ記事の品質が低い | Antigravityに「記事の品質を上げるプロンプトに修正して」 |
| noteが売れない | 無料記事を1-2本出して流入を増やす |
| アフィリエイト審査落ちた | 記事数を増やして再申請（3ヶ月後） |
| 新しい記事が欲しい | Antigravityに「○○についてブログ記事書いて」 |
| コンテンツネタがない | `sns/x_auto_poster.py` のTOPIC_IDEASを参考にする |
